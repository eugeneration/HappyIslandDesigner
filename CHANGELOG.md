# Changelog

All notable changes to Happy Island Designer are documented in this file.

## [vX.X.X]

## [v2.0.1] - 2026-03-11

- Add settings modal with language picker
- Fix modal overlay requiring two clicks to dismiss (AppModal wrapper)
- Add translation disclaimer for non-English languages
- Add Korean, French, and German translations
- Translate remaining hardcoded strings for i18n

## [v2.0.0] - 2026-03-06

- New island setup wizard with tile-based map editor
- Generate island from screenshot
- V1 to V2 map conversion tool
- SVG-based edge tiles replacing raw tiles
- New map icons based on minimap
- Stairs/bridges rotation and extension
- Add Japanese, Chinese (Simplified/Traditional), and Spanish translations
- Loading skeleton and performance optimizations
- GA4 analytics
- SEO improvements
- Bundle splitting and lazy loading

## [v0.5.0] - 2026-01-15

- Upgrade to webpack 5 and TypeScript 5
- Fix react-modal compatibility
- Show cmd instead of ctrl for macOS hotkeys

## [v0.4.1] - 2020-05-16

- Fix screenshot target movement and zoom
- Add island layouts and blank map option
- Mobile scrolling improvements
- Add all in-game flowers
- Community-contributed island layouts

## [v0.4.0] - 2020-04-19

- TypeScript rewrite of codebase
- Scroll and zoom controls for touchpad/mouse
- i18n support with Japanese, Spanish, and Chinese (Traditional) translations
- Screenshot overlay for tracing
- iOS/Safari file saving fixes
- HEIC to PNG conversion for screenshots
- Vendor bundle splitting

## [v0.3.0] - 2020-04-06

- Screenshot tracing overlay (upload, perspective correct, overlay on map)
- Mobile screenshot improvements
- Fix save encoding for empty objects

## [v0.2.1] - 2020-03-28

- Adjust object sizes for bridges, shops, and town square

## [v0.2.0] - 2020-03-17

- Undo/redo buttons
- Flexible menu system overhaul
- Object deletion button
- Brush size UI controls
- Compressed map save format (lz-string)
- Many new object sprites (flowers, bridges, trees)
- Mobile improvements

## [v0.1.1] - 2020-03-12

- Color picker hotkey
- Various small fixes

## [v0.0.1] - 2020-02-24

- Initial release: drawing, layers, grid, sprites, undo/redo, zoom/pan
